# Prompt2Widget

**Создай любой интерактивный веб-виджет за секунды — просто опиши его словами.**

Prompt2Widget — это полностью локальная ИИ-платформа, которая превращает текстовое описание в готовый, функциональный и интерактивный веб-виджет (HTML + встроенные CSS и JS).

От калькулятора цен до умного рекомендателя тарифов — описываешь задачу на русском языке, получаешь рабочий код мгновенно.

## Возможности

- Генерация полноценного виджета по текстовому промпту
- Полностью самодостаточный код (один HTML-файл со стилями и скриптами внутри)
- Встроенный интерактивный редактор кода прямо в чате — правишь код, сразу видишь результат
- Живой предпросмотр в реальном времени (debounced 500 мс)
- Два ИИ-ассистента на выбор:
  - **GigaChat** (через официальное API Сбера) — отличная поддержка русского, стабильно
  - **Qwen 2.5-Coder 4B** (локально через EXL2) — молниеносно, без интернета и лимитов

## Как запустить локально (3–5 минут)

```bash
# 1. Клонируем репозиторий
git clone https://github.com/izhdenevav/Prompt2Widget.git
cd Prompt2Widget

# 2. Устанавливаем зависимости
pip install -r backend/requirements.txt

# 3. Запускаем сервер
uvicorn main:app --host 127.0.0.1 --port 8000 --reload

Откройте в браузере: http://127.0.0.1:8000
Готово! Работает полностью офлайн (если используете Qwen) или с API-ключом GigaChat.
```
